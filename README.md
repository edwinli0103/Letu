
# letu projuct
## 宠物电商demo

### 项目简介

- 整体框架  vue.js
- 路由   vue-router
- 异步数据请求 axios
- 全局状态管理 vuex
- 辅助ui框架 vux

### 环境要求

- node版本4.0以上
- npm 3.0 以上 或者 cnpm

### Build Setup

``` bash
# 项目初始化
npm install

# 构建项目和 启动服务 localhost:8080
npm run dev

# 构建用于生产 (生成压缩版项目文件)
npm run build

# 构建用于生产并查看包分析器报告
npm run build --report
```
